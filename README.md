**Как обновить данные по дашборду?**

**Шаг 1.** Скачай файлы `mfm-2026_daily_report.ipynb`, `russian_fo.py` (список федеральных округов), `macroregions.py` (список макрорегионов)
В этом файле хранится код, который поможет сформировать выгрузку. Для этого тебе понадобится Cursor, PyCharm или Anaconda.
Если не знаешь что это и как это установить – придется гуглить или просить объяснения у ИИ.

**Шаг 2.** Изменить в ячейке загрузки данных в файл `mfm-2026_daily_report.ipynb` наименования файлов и путь до них (формируются 3 выгрузки: заявки, эссе, видеовизитки)

**Шаг 3.** Прогнать все ячейки и получить итоговую выгрузку (в последней ячейке нужно изменить путь сохранения файла)

**Шаг 4.** Полученный файл с данными загрузить в папку `data`
Данный шаг можно пропустить и перейти к **Шаг 5**.

**Шаг 5.** Перейти в дашборд по ссылке и выбрать из фильтра сегодняшний файл или внести файл вручную при помощи специальной кнопки
В репозитории два подходящих файла: `index.html` и `Прирост за выходные.html`. `Прирост за выходные.html` был создан для того, чтобы показать прирост за выходные.
Чтобы показать прирост за выходные в дополнительном поле выбери дату, с которой необходимо сравнить сегодняшний день. Например, сегодня 09 марта. Соответственно, мы выбираем 06 марта для сравнения.

**Шаг 6.** Сохранить файл в pdf (я делаю это при помощи комбинации Ctrl+P/Cmd+P)

**ВАЖНО**
Обращай внимание, что красная дата соотвествует сегодняшней при скачивании файла.
